# 淨資產退休試算器

暗色主題、單機可跑的退休試算 Web App。

## 功能

- 📊 多計畫管理：可建立、複製、重命名、刪除多組計畫
- 📅 年度逐年規劃：左側邊欄切換年份，每年獨立資料
- 💰 收入與投資：年收入、投資部位、報酬率、質押利率、理財型利率
- 📉 月支出項目：**每項可獨立設定生效月份數（預設 12）**，例如某月貸款只繳 5 個月
- 🏗️ 年度固定支出：自由新增/刪除（工程款、裝潢…）
- 📊 即時計算：年報酬、淨現金流、年末投資部位、年末淨資產
- 📈 模擬到目標：自動推算每年走向，找出達成 4000 萬（或自訂）的年份
- ⚖️ 計畫比較：勾選多個計畫並排顯示達成年/達成率/年度淨資產走勢
- 💾 SQLite 持久化：所有資料存於 `planner.db`

## 快速啟動

```bash
cd apps/retirement-planner
pip install -r requirements.txt
python app.py
```

預設於 <http://127.0.0.1:5050> 啟動。首次啟動會自動建立 `planner.db` 並預載
2026–2030 範例資料（年收入 280 萬、月支出合計 30.1 萬、投資 2561 萬）。

## 單位

所有金額單位皆為 **萬元**（台幣）。比率以百分比輸入，例如 6 = 6%。

## 計算公式

```
總月支出 (年化) = Σ(月支出金額 × 月份數)
總年支出       = 總月支出 + 年度固定支出
年報酬         = 投資部位 × 報酬率
質押成本       = 投資部位 × 質押利率
理財型收益     = 投資部位 × 理財型利率
淨現金流       = 年收入 − 總年支出 − 質押成本 + 理財型收益
年末投資部位   = 投資部位 + 年報酬 + 淨現金流
年末淨資產     = 期初淨資產 + 年報酬 + 淨現金流
```

模擬時，已輸入的年度使用實際資料；超出年度後沿用最後一年的設定推算下去。
